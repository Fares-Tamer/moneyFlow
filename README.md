# 🥗 GainFlow

A modern calorie tracking dashboard built with React and Tailwind CSS.

## ✨ Features

- Calculate your daily calorie goal
- Track meals
- Add / Edit / Delete meals
- Filter meals by category
- Automatic calorie summary
- Local Storage persistence
- Responsive UI
- Profile page
- Goal setup page

## 🛠 Tech Stack

- React
- Tailwind CSS
- React Router
- Local Storage
- Lucide React
